# ms-dacs2025-bff 

Microservicio de backend for frontend.

## Descripción

Este proyecto es el Backend For Frontend (BFF) de la aplicación principal. Su función es actuar como una capa intermedia entre el frontend y los distintos servicios del backend, centralizando la lógica de integración, autenticación y autorización, simplificando el consumo de APIs y mejorando la seguridad y la experiencia del usuario.
